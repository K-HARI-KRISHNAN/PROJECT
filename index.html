Alert Vision

Advanced Camera Security & Intrusion Alert System (Working Model) Alert Vision is a fully functional security alert system designed to provide real-time protection against tampering, intrusion, or attempts to disable surveillance cameras. This working model uses smart sensors and communication modules to immediately detect threats, alert emergency contacts, and even defend the secured area. ⚙️ Key Features: 📸 ESP32-CAM for Live Capture: The system uses an ESP32-CAM module to monitor the environment. If a threat is detected, it automatically captures an image and stores or transmits it for verification. 🪓 Vibration Sensor for Tamper Detection: If anyone attempts to break or shake the camera, a vibration sensor detects the movement, and the system triggers an alarm, captures an image, and begins the response process. 🕵️‍♂️ IR Sensor for Obstruction Detection: If someone tries to hide the camera's view using an object or cover, the IR sensor detects reduced visibility and activates the alert system. 📣 Buzzer Alert System: A loud buzzer is activated to warn the intruder and notify nearby individuals of a security breach. 📞 Emergency Call and SMS Alert: The system is integrated with a GSM module or similar method to send an emergency SMS and/or call to pre-defined contacts instantly when an intrusion is detected. ⚡ Stun Gun Module for Door Protection: A stun gun power module is installed near the door handle. If someone tries to forcefully open the door, the system delivers a shock as a deterrent—designed for high-security zones (with proper safety handling in real-world usage).

Audio Analyzer

The Audio Analyzer is a standalone prototype developed to analyze sound signals in real-time. This system is capable of detecting, processing, and visualizing audio data for applications such as noise monitoring, voice recognition, environmental sensing, and educational demos. Key Features: 🎙️ Microphone-Based Audio Input: The analyzer uses a sensitive microphone sensor (like the MAX9814 or KY-038) to capture surrounding audio signals for analysis. 📊 Real-Time Audio Visualization: With an ESP32 microcontroller, the system processes audio input and displays it on an OLED screen or through serial plotting, helping visualize sound levels, waveforms, or frequency spectrums. 🎛 Volume Level Detection: The device can detect and display the volume intensity (amplitude) in real time, making it useful for noise level monitoring or as a VU meter. 📡 FFT and Frequency Analysis (Optional Extension): Using Fast Fourier Transform (FFT), the system can break down audio into frequency components to identify pitch, tone, and frequency bands—great for music, voice analysis, or machine sound monitoring. 🔔 Threshold-Based Alert System: You can configure it to trigger an LED, buzzer, or message when sound exceeds a set volume, making it useful for alert systems in quiet zones or industrial environments.

Borewell Rescue Robot

This project is a prototype of a Borewell Rescue Robot, designed to rescue children who accidentally fall into borewells. Such incidents are extremely dangerous due to the narrow space, limited oxygen, and delayed rescue operations. Our solution aims to minimize rescue time and enhance safety using a compact, intelligent robotic system. Key Features: 🔹 Adjustable Mechanism: The robot is designed to fit and move inside pipes of various diameters. It automatically adjusts itself to the borewell's size, ensuring smooth and safe descent. 🔹 ESP32-CAM for Live Video: An ESP32-CAM module is mounted on the robot to provide real-time video streaming, allowing rescuers to visually monitor and control the robot from the surface. 🔹 Child Detection and Automatic Claw System: Equipped with sensors, the robot can detect the presence of a child inside the borewell. Once detected, an automatic claw mechanism activates to gently and securely grasp the child for lifting. 🔹 Safe and Precise Operation: The robotic claws are designed to minimize injury risks, offering a gentle yet firm grip. The system is built to handle confined spaces and ensure stability during operation. 🔹 Prototype Focus: This model is a working prototype to demonstrate the rescue concept. It emphasizes mobility inside pipes, automatic detection, gripping mechanics, and live monitoring.

Traffic Rider Game

￼​ Traffic Rider Game – Real-Time Steering Controlled Car Game (Working Model) Traffic Rider Game is a fully functional gaming setup where players control a car using a physical steering wheel. The game tests reflexes and coordination, providing a fun and engaging experience. It’s a combination of mechanical motion, visual simulation, and sensor-based collision detection! ⚙️ Key Features: 🛞 Steering Wheel for Car Movement: The player uses a real steering wheel to move the center car image left and right on the screen or chart, simulating actual driving. 🛣️ Moving Chart with Obstacle Cars: A moving chart (roller mechanism) is installed in the background, constantly scrolling to simulate a road filled with other car images. This creates the effect of moving traffic. 💥 Collision Detection using Color Sensor: A color sensor is mounted to detect collisions between the center car and obstacle cars printed on the moving chart. If a collision is detected (i.e., when the sensor detects a different color indicating another car), The game automatically ends (the system can stop the chart movement, play a sound, or turn off the screen). ⚡ Speed Controller: The chart's movement speed (i.e., the traffic speed) is adjustable through a speed controller, allowing players to set different difficulty levels: Slow (easy mode) Medium (normal mode) Fast (hard mode) 🔔 Game Over Mechanism: When a collision is detected: The roller chart stops moving. A buzzer can be activated to indicate a crash. The system can display "Game Over" visually or reset for a new game.
